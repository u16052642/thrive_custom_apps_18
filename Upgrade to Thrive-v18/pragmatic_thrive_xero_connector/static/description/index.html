Thrive XERO Connector Features

Pragmatic Thrive XERO Connector is an Thrive App which does bi-directional integration between Xero and Thrive

Xero Accounting Interface


Step 1: Go to https://developer.xero.com and click on My Apps and login to xero if you have not logged in. Fill username and password and click on signinbutton (If you are already have an Account or else continue with Signup process)





Step 2: After successful login, click New Apps button and create a Public App.



Step 3: Select the created App, you will get all the details from the page.



Thrive Interface

Step 4: Sign In to Thrive Admin Account -> Go to Settings -> Companies

Step 5: Click on company name as per requirement -> Select Xero Accounting Menu



Step 6: Inside Xero configuration you can see two different menus named as Import, Log Details and Credentials. Click on Credentials.

Paste the credentials for Client Id, Client Secret copied from Xero to respective fields in Thrive.



Step 7: There are two ways to do the Authentication.

  1. Click on Test Connection and you will be redirected to a new page (allow popup windows for your browser) where you will find the verifier code, copy the code from there and paste it in the verifier code field in thrive and then click onAuthenticate.
  2. Click on AutomatedAuthentication.

Authentication :-This will check the credentials from both the sides and if it matches, authentication gets successful, only then it will give permission to sync the data.

Step 8: Xero Configuration:

Inside Xero configuration you can see two different menus named as Import and Credentials.
When you click on Import, you can see all the data fields available for Sync operation. Select specific option as per your requirement.
