HR Employee Overtime Requests with Payslip Thrive Apps

Employee Overtime Requests with Payroll Thrive apps will help employees to request overtime salary payment and HR manager to manage overtime requests. This app will help the HR Manager or Department Manager to create employee overtime request on behalf of multiple employees using single request.

Features

Allows employee to request for overtime requests.

App offers employee to create overtime request.

Multiple Overtime Request.

HR Manager or Department Manager can create overtime request on behalf of multiple employees.

HR Manager and Department Manager Approval.

To submit the overtime request HR Manager and Department Manager Approval is required.

Integrate Overtime Requests in Employee Payslips.

This module will use fixed cost(hours) on salary rule. Means you have to use fixed cost price / hours to calculate overtime allowance.
Demo Request Video Tutorial

Overtime User Access

  • HR(Overtime Salary)
    • Users with this access rights can approve employee overtime request.
  • Department Manager(Overtime Salary)
    • Users with this access right is responsible for approve and refuse employee overtime request by department. Also can approve multiple overtime request.
  • Show Own Request
    • Users with this access rights can show only their own overtime requests.

Overtime Request

The figure shows the form view of Overtime Request(Employees > Overtime >Overtime Request).

With this form employee can create overtime request and submit it to HR Manager and Department Manager for approval.

Multiple Overtime Request

The figure shows the form view of Multiple Overtime Request(Employees > Overtime >Multiple Overtime Request).

With this form HR Manager or Department Manager can create overtime request, multiple employees can be selected on form and approval flow will be as it is like single overtime request form.

Overtime Salary Rule

Salary Structure

You have to manually add overtime salary rule on your employee salary strucuture which will allow integration between overtime request and payroll using salary rules.
