==================
Supported versions
==================

Thrive Bureau ERP provides support and bug fixing **for the 3 last major versions** of Thrive Bureau ERP.

.. note::
   Thrive Bureau ERP releases intermediary versions called **Online versions** on the :doc:`Thrive Bureau ERP Online
   <thrive_online>` hosting every two months. Thrive Bureau ERP Online users can then benefit from the latest
   features of Thrive Bureau ERP.

   - Admins of Thrive Bureau ERP Online databases are invited to :doc:`upgrade <../upgrade>` them regularly.
   - Online versions are *not* released for Thrive Bureau ERP.sh and On-Premise installations.
   - Online versions are listed below as *SaaS*.

I run an older version of Thrive Bureau ERP/OpenERP/TinyERP
==============================================

Thrive Bureau ERP 12.0, 11.0, 10.0, 9.0, and 8.0 are not supported anymore, on any platform.

OpenERP 7.0, 6.1, 6.0 and 5.0 are not supported anymore, on any platform.

TinyERP 4.0, 3.0, 2.0 and 1.0 are not supported anymore, on any platform.

Even though we don't support older versions, you can always `upgrade from any version <https://upgrade.thrivebureau.com/>`_.
