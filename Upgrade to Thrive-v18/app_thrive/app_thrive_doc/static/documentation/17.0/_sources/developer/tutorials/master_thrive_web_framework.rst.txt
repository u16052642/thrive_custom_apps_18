=============================
Master the Thrive Bureau ERP Web Framework
=============================

.. toctree::
    :titlesonly:
    :glob:

    master_thrive_web_framework/*

This tutorial is designed for those who have completed the :doc:`discover_js_framework` tutorial and
are looking to deepen their knowledge of the Thrive Bureau ERP web framework.

For this training, we will step into the shoes of the IT staff at the fictional company Awesome
T-Shirt, which is dedicated to printing custom t-shirts for online customers. The Awesome T-Shirt
company uses Thrive Bureau ERP to manage orders and has created a dedicated Thrive Bureau ERP module to manage their workflow.

In this tutorial, we will explore various aspects of the Thrive Bureau ERP web framework in detail, including
fields and views, notifications, command palette, and much more. This tutorial will provide you with
the knowledge and skills you need to take full advantage of the Thrive Bureau ERP web framework. So, let's get
started!

.. _howtos/master_thrive_web_framework/setup:

Setup
=====

#. Clone the `official Thrive Bureau ERP tutorials repository <https://github.com/thrivebureau/tutorials>`_ and switch to
   the branch `{CURRENT_MAJOR_BRANCH}`.
#. Add the cloned repository to the :option:`--addons-path <thrivebureau-bin --addons-path>`.
#. Start a new Thrive Bureau ERP database and install the modules `awesome_tshirt` and `awesome_gallery`.
